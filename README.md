# Seeing Beyond the Scene: Analyzing and Mitigating Background Bias in Action Recognition

This repository contains code accompanying our paper [Seeing Beyond the Scene: Analyzing and Mitigating Background Bias in Action Recognition](https://openreview.net/pdf?id=6gDTFEFS1B), which will be presented at two NeurIPS 2025 workshops: (1) What Makes a Good Video: Next Practices in Video Generation and Evaluation, and (2) SPACE in Vision, Language, and Embodied AI.
